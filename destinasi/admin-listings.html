Loko Coffe Shop


Alamat :

No.3 A, Jl. Ps. Kembang, Sosromenduran, Gedong Tengen, Kota Yogyakarta, Daerah Istimewa Yogyakarta 55271


Detail :

Loko Coffee adalah kafe, restoran atau tempat makan yang dimiliki oleh PT Reska Multi Usaha yang merupakan anak perusahaan dari PT Kereta Api Indonesia. Loko Cafe ini dikonsepkan untuk meningkatkan kenyamanan para calon penumpang kereta api pada saat menunggu keberangkatan di stasiun. Tidak cuma para calon penumpang saja yang boleh berkunjung disini, masyarakat umum yang tidak naik kereta api atau bertiket pun boleh berkunjung disini. Loko Cafe ini buka setiap hari mulai dari jam 06.00-22.00 WIB.
